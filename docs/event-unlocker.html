Event Rewards Unlocker

Rewards for Happy at Home Login Event were stored locally on your computer. Because of that some people lost them, and so EA decided to add them all to the base game in update version 1.109.207. So you no longer need this tool. Rewards for Reaper's Rewards Event are stored on a server. That means unlocking them would work in offline mode only. Every time you start the game online it overwrites a file with reward unlocks. Given all that you should use mods instead - they work both online and offline.

To quickly find the event rewards in game you can use the "Special" > "Event Rewards" filter.
